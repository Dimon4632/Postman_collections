Для выполнения тестов в папках: Get List By User, Get List By Tag и Get Post by id неоходимо 
извлечь из тела ответа GET Get List в отдельные JSON файлы: id owner для Get List By User, 
tags для Get List By Tag и id, для Get Post by id. После чего выполнить Run для кажой папки 
с добавлением соотверствующего Data файло (JSON ).


Документация: https://dummyapi.io/docs
